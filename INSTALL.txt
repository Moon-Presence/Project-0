Для сборки программы  необходимо установить Java SE Development Kit 8u202 который доступен 
по ссылке https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

Команды которые надо выполнить что бы собрать проект
для сборки запустить скрипт build.bat.

Для запуска программы Tetris необходимо установить Java SE Runtime Environment 8u202 который доступен по ссылке https://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html

Для запуска программы использовать скрипт Run.bat.
